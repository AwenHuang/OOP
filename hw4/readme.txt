<makefile>
	輸入make => 產生hw4執行檔
	輸入make clear => 清除hw4執行檔
<hw4.h>
	將3個所要求的class打在此檔案(Point,Line,Convexhull),另外還有幾個自己所定義的函式

<hw4.cpp>
	將hw4.h所定義宣告的函式內容打在此檔案(Point,FindLine,isOnRight,FindConvexHull_bf,FindConvexHull_jm,FindLowestPoint,ComputeAngle,FindInternalPoint,一些額外的compare函式)

<main.cpp>
	讓使用者可以選擇要使用暴力法或是Javis演算法
	印出所花的時間(us)，將結果輸出到Output.txt
